## Project Overview
This project predicts diabetes risk using clinical and lifestyle health indicators such as BMI, glucose, blood pressure, insulin, diet, physical activity, sleep, and other behavioral factors.

## Dataset
- File: Healthcare.csv
- Rows: 18,000
- Columns: 30
- Target Variable: Outcome (0 = No Diabetes, 1 = Diabetes)
